Welcome to my personal [Scoop](https://scoop.sh) bucket! 🌟 Just like Doraemon's four-dimensional pocket, you'll find all sorts of magical and handy tools here for your Windows environment. ✨

## 🚀 How to use

It's super easy to get started! Just open your PowerShell and cast these spells:

```pwsh
# Add the magical pocket to your scoop
scoop bucket add DoraemonPocket https://github.com/AndeFriday/DoraemonPocket.git

# Pull out a shiny new gadget! 🎁
scoop install DoraemonPocket/<app-name>
```
